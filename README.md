# ❤️ Heart Disease Prediction Using Machine Learning

## 📌 Overview
This project focuses on predicting the likelihood of heart disease using Machine Learning techniques in Python. The dataset was analyzed, preprocessed, and used to train classification models for predicting heart disease risk based on medical attributes.

---

## 🎯 Objective
- Analyze heart disease dataset
- Perform data preprocessing and visualization
- Train machine learning classification models
- Predict the presence of heart disease
- Evaluate model performance

---

## 🛠️ Tools Used
- Python
- pandas
- numpy
- matplotlib
- seaborn
- scikit-learn

---

## 📊 Dataset
The dataset contains medical information used to predict heart disease.

### Features Included:
- Age
- Sex
- Chest Pain Type
- Blood Pressure
- Cholesterol
- Maximum Heart Rate
- Blood Sugar
- ECG Results
- Exercise Induced Angina
- Oldpeak
- Thalassemia

### Target:
- Presence or absence of heart disease

---

## 🤖 Machine Learning Models
The following models were used for prediction:

- Logistic Regression
- Random Forest Classifier
- Decision Tree Classifier

---

## 🔍 Analysis Performed
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Correlation analysis
- Data visualization
- Model training and testing
- Prediction evaluation
- Accuracy comparison between models

---

## 📈 Evaluation Metrics
The models were evaluated using:

- Accuracy Score
- Confusion Matrix
- Classification Report

---

## 🧠 Key Insights
- Dataset was properly structured for classification tasks
- Some medical features showed strong correlation with heart disease
- Random Forest provided better prediction accuracy
- Data visualization helped identify important patterns
- Feature analysis improved model understanding

---

## ✅ Conclusion
This project demonstrated how machine learning can be used to predict heart disease based on medical data. Among the implemented models, Random Forest performed better and provided more accurate predictions compared to other classification models.
